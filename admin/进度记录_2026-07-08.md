- ✅ JwtAuthenticationFilter: 校验 Token，设置 SecurityContext

- ✅ 未登录请求返回 401
- ✅ 带Token请求正常访问

**关键技术点：**
- JwtAuthenticationFilter 必须设置 `SecurityContextHolder.getContext().setAuthentication(authentication)`
- 不能只设置 request.setAttribute()，否则Spring Security会拒绝请求

### 测试命令
```powershell
# 登录获取Token
$resp = Invoke-RestMethod -Uri "http://localhost:8080/api/sys/user/login" -Method POST -ContentType "application/json" -Body '{"username":"admin","password":"123456"}'
$token = $resp.data.token

# 测试接口
$headers = @{ token = $token }
Invoke-RestMethod -Uri "http://localhost:8080/api/dashboard/stats" -Headers $headers
```
